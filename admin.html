미설정

외부 API · 기상 (침수경보)

기본적으로 기상청 매분관측 페이지(weather.go.kr)에서 강수15M·60M·6H를 수집합니다. 공공데이터 API는 선택 사항이며, 매분관측 조회 실패 시에만 사용합니다.

기상청 강우 API

CSV의 기상관측소코드(stn / stnIds) 기준으로 강우량을 조회합니다. 코드가 없으면 업로드 시 좌표·사고위치로 ASOS 지점번호를 자동 매핑합니다.

미설정

외부 API · 뉴스

침수경보 대시보드 우측 패널에 표시할 기상·재난 뉴스 API와 검색 키워드를 설정합니다.

뉴스 검색 API (네이버 비로그인 방식)

네이버 검색 API는 OAuth 없이 HTTP 헤더(X-Naver-Client-Id, X-Naver-Client-Secret)만으로 호출합니다. 서버가 자동으로 붙입니다.

인증키는 ClientId|ClientSecret 형식으로 입력하세요.

브라우저 주소창에 URL만 열면 query·헤더가 없어 400 오류가 납니다. 관리자 설정 저장 후 대시보드에서 확인하세요.

미설정

뉴스 검색 키워드

쉼표로 구분해 입력합니다. 키워드별로 검색 후 중복을 제거해 대시보드에 표시합니다.

예: 철도역 침수, 철도시설 침수, 철도 침수 사고, 지하철역 침수

외부 API · 문자(SMS)

자율보고 민원 배정·이첩 시 담당자가 문자 발송 버튼으로 알림을 보냅니다. API는 POST JSON { to, message } 형식을 가정합니다.

문자(SMS) 알림 API

Authorization Bearer 헤더에 API Key를 붙여 호출합니다.

미설정

자율보고 접속 주소

2차 담당 배정·계정 안내 문자에 포함됩니다. 비우면 서버 주소 + /dashboard/self-report 가 사용됩니다.
